CI/CD Pipeline Project

Automating Deployment with AWS CodePipeline, CodeBuild, and CodeDeploy

Project Overview
This project automates application deployment using AWS CI/CD services. By integrating CodePipeline, CodeBuild, and CodeDeploy, the project ensures smooth and automated deployment to EC2 or ECS environments. Docker images are built and deployed with notification alerts.
